# 📊 Global Superstore Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** built using the **Global Superstore 2016** dataset. The dashboard provides business insights into sales performance, profitability, customer behavior, product performance, shipping efficiency, and return analysis across multiple global markets.

The dashboard is designed to help business stakeholders monitor key performance indicators (KPIs), identify sales trends, compare regional performance, and support data-driven decision-making.

---

# 🎯 Project Objectives

* Monitor overall business performance using KPIs.
* Analyze sales and profit trends over time.
* Compare regional and geographical performance.
* Identify valuable customer segments.
* Evaluate product category profitability.
* Investigate return rates and shipping performance.

# 📊 Key Findings

* Sales exceeded **£12 million** with a healthy overall profit.
* Technology products generated the strongest profit margins.
* Consumer customers contributed over half of total revenue.
* Holiday seasons delivered the highest sales performance.
* Large discounts significantly reduced profitability.
* Some regions experienced higher return rates than others.
* A data relationship issue caused missing manager information for Canada due to inconsistent region names.
